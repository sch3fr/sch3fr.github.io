deSnap GTA Tool

This is a beautiful example of creating a solution for a real problem, not coding for the sake of it. deSnap is a tool for extracting pictures from the Snapmatic feature in Grand Theft Auto Online. The images are stored on a drive, but in binary unopenable file. This tool strips the binary of useless header data and exports the images to the desktop. Simple as that. And the code is beautiful with less than 50 lines. Read more here or on Github
